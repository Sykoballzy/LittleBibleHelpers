# Resources — the art drop zone

Save generated PNGs into THIS folder with the exact filenames from
`ART_SPEC.md` (e.g. `art_lion.png`). Then tell Claude — the files get
committed and pushed, and the app automatically uses each image in place of
its vector placeholder. No code changes per asset.

Rules:
- Exact, case-sensitive filenames: `art_<key>.png` (see the manifest).
- PNG with a REAL transparent background (not white).
- 1024×1024 for sticker art; 2048×1536 for backgrounds.
- One subject, centered, ~8% padding on all sides.
